## 🎯 Comandos Principais

### Windows PowerShell
```powershell
cd docker
.\build.ps1 -Version "1.0.0"
docker-compose -f docker-compose.yml up -d
curl http://localhost:5000/health
```

### Linux/Mac
```bash
cd docker
./build.sh 1.0.0
docker-compose -f docker-compose.yml up -d
curl http://localhost:5000/health
```

### Make (Recomendado)
```bash
cd docker
make help         # Ver todos
make build VERSION=1.0.0
make up
make health
```
