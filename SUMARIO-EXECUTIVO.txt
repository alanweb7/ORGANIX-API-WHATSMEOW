## 🚀 Como Usar

### Opção 1: Desenvolvimento Local (SEM mudança)
```bash
cd api
go run .
```
✅ **Funciona igual!**

### Opção 2: Docker com Make (recomendado)
```bash
cd docker
make build VERSION=1.0.0
make up
make health
```
✅ **Mais fácil!**

### Opção 3: Docker Direto
```bash
cd docker
docker-compose -f docker-compose.yml up -d
```
✅ **Funciona!**

### Opção 4: Menu Interativo
```bash
# Windows
cd docker && .\start.cmd

# Linux/Mac
cd docker && ./start.sh
```
✅ **Amigável!**
