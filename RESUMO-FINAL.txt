    ────────────────────────────────────
    cd api
    go run .
    
    ✅ Igual como era antes! Nada mudou.


2️⃣  COM DOCKER (NOVO - MAIS ORGANIZADO)
    ──────────────────────────────────────
    cd docker                    ← Entre aqui primeiro
    
    Opção A - Usando Make (recomendado):
    make help                    ← Ver todos os comandos
    make build VERSION=1.0.0
    make up
    make health
    
    Opção B - Docker Compose direto:
    docker-compose -f docker-compose.yml up -d
    
    Opção C - Menu interativo:
    .\start.cmd                  (Windows)
    ./start.sh                   (Linux/Mac)


3️⃣  COMMANDOS RAPIDOS (Copy & Paste)
    ────────────────────────────────
    # Windows PowerShell
    cd docker
    .\build.ps1 -Version "1.0.0"
    docker-compose -f docker-compose.yml up -d
    curl http://localhost:5000/health
    
    # Linux/Mac bash
    cd docker
    chmod +x build.sh start.sh
    ./build.sh 1.0.0
    docker-compose -f docker-compose.yml up -d
    curl http://localhost:5000/health


4️⃣  DEPLOY EM PRODUCAO
    ──────────────────
    cd docker
    ./deploy.sh seu-usuario@seu-vps v1.0.0
